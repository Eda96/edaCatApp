# edaCatApp
Description →
With this app you can find cat photos and descripitions about cats from all over the world. 

Must Have →
You can get cat photo.
You can see as many cat photos as user chooses from three options.
You can choose special cat breeds and get information.

Should have →
You can reach wikipedia for detailed information.
