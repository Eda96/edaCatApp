# edaCatApp
Description
With this app you can find cat photos and descripitions about cats from all over the world. 

Must Have
The user can get cat photo.
The user can see as many cat photos as user chooses from three options.
The user can choose special cat breeds and get information.

Should have
The user can reach wikipedia for detailed information.
